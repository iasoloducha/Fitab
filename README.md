# FITLAB Gym Castelar

Sitio web del gimnasio FITLAB en Castelar, Buenos Aires.

## Características

- Landing page institucional
- Generador de rutinas de entrenamiento (exporta a PDF)
- Planes de membresía (Basic, Pro, Elite)
- Información de contacto y ubicación

## Estructura

```
├── index.html    # Estructura HTML
├── styles.css    # Estilos
├── scripts.js   # JavaScript
└── fitlab-web/  # Imágenes
```

## Tech Stack

- HTML5
- CSS3 (sin framework)
- Vanilla JavaScript

## Desarrollo

Abrir `index.html` en un navegador.
